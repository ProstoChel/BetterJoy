-Настройка сочетаний клавиш
-Сброс сопротивления атомайзера (актуально для ТК)
-Some bugfixes

Управление по умолчанию (старался сделать комфортным)
Главный экран:
2Fire - температурная доминанта вкл/выкл в ТК режиме / прехит в вариватте
3Fire - выбор профиля
4Fire - редактирование экрана

+-Fire (три кнопки) - изменение яркости экрана
+-Fire (долгое) - вызов фонарика (яркий экран с белыми пикселями) // будет реализовано

-Fire - меню
-Fire (долгое нажатие) - переворот экрана
-Fire (ещё дольше) - обмен кнопок +- . Как правило, переворот экрана требует ещё и обмен клавиш // в будущем

+Fire - стелс режим
+- блокировка клавиш

Меню:
-Fire - подняться на уровень вверх
+- - выход из меню



Отличия от Sur Myevic
- При парении ТК обновляется регулярно, а не раз в 2 секунды.- Убран пэкмэн, ползунок затяжки.
- Уменьшена задержка срабатывания нажатия на Fire с 60 до 10 мс.
- Яркость регулируется не шагом в 2%, а по 1/3. (0-33-66-100. Яркость изменяется быстро и любой кнопкой (+-) можно указать любое значение
- Замена клавиш +- перенёс в раздел Interface
- Добавлен пункт выхода из Прехита
- Оптимизировано управление (уменьшены задержки, долгое нажатие Fire не 2, а 0.5 секунды, подробности в инструкции)
- Интерфейс (основной экран) больше напоминает ArcticFox
- Множество других улучшений интерфейса
- При показе времени тяги экран не зависает, а в режиме ТК показывает живую температуру
Отличие от ArcticFox
Поддержка сочетания клавиш +-Fire (спасибо MicroSur)
-Лёгкий стелс - экран гаснет не сразу при нажатии Fire, а после гашения экрана
-Базовая прошивка намного более функциональна, чем AF, но требуется их доработка. Есть игры, режим ByPass, куча других возможностей
